Dúvidas Frequentes

Meu notebook só tem 2GB de RAM. Funciona?
Sim! O foco do curso é justamente hardware limitado. Usamos distribuições Linux específicas (como Antix ou MX Linux) que consomem menos de 300MB de RAM ao iniciar, deixando o resto livre para você usar.
Preciso ser técnico para fazer isso?
Não. O curso é desenhado para usuários comuns. Se você sabe usar um pendrive e seguir instruções em vídeo, você consegue. Não usamos termos complicados sem explicar.
Tenho medo de abrir o notebook. É obrigatório?
Não é obrigatório. O Módulo 1 ensina a abrir para limpeza e upgrade, mas se você não quiser fazer upgrades físicos, pode pular direto para o Módulo 3 e apenas instalar o sistema Linux, o que já resolve 70% da lentidão.
Funciona para Desktop (PC de mesa)?
Sim, totalmente. A lógica de escolha do sistema, instalação e otimização é idêntica. Só muda a forma física de abrir o gabinete, mas o software é o mesmo.
